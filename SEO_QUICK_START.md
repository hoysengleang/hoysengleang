## Emergency Checklist

### If Rankings Drop
1. Check for technical errors
2. Review recent changes
3. Verify sitemap submission
4. Check mobile usability
5. Analyze competitors
6. Review Google updates

### If Site is Slow
1. Run Lighthouse audit
2. Optimize images
3. Check hosting performance
4. Review code changes
5. Enable caching
6. Minimize JavaScript

### If Not Indexed
1. Submit sitemap again
2. Check robots.txt
3. Verify no-index tags
4. Request manual indexing
5. Check for penalties
6. Review structured data
